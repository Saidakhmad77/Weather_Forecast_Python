# Weather Forecaster

## Overview
Weather Forecaster is a simple web application that provides real-time weather information for any city. Users can enter a city name to receive the current weather conditions and temperature, along with an appropriate weather icon. The application features a modern, user-friendly interface with a dark theme for improved visibility.

## Features
- Search for weather information by city name.
- Displays current weather conditions and temperature.
- Utilizes weather icons to visually represent weather conditions.
- Responsive design for mobile and desktop users.

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API for weather data

## Getting Started
1. Clone the repository:  
   `git clone https://github.com/Saidakhmad77/Weather-Forecaster.git`
2. Open `index.html` in your web browser.
